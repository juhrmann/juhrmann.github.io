Motivation
==========
Docker-Container werden zunehmend beliebter bei der Verteilung von Software und beim cloudbasiertem Betrieb von Softwaresystemen.
Dabei werden vorgefertigte (Base-)Images verwendet, auf denen die Anwendung aufgebaut ist.
Meist werden wenig Aussagen über die Sicherheit dieser Docker-Images getroffen.

Aufgabenstellung
================
Sie entwerfen ein Evaluierungssystem, das Sicherheitskriterien definiert.
Anhand von frei verfügbaren Docker-Images testen Sie, ob diese Sicherheitskriterien erfüllt sind.

Alternative oder Ergänzung
--------------------------
Sie erstellen ein Messsystem als Docker-Image, das
* als Image allgemein zur Verfügung gestellt wird und meldet, wenn es verwendet wird, oder
* das Sicherheitsniveau der Docker-Umgebung analysiert und automatisiert einen Bericht dazu erstellt.

Hinweise
========
* Interesse an IT-Sicherheitsthemen wird vorausgesetzt.
* Sie sollten in der Lage sein, einfache Tests zu automatisieren.
